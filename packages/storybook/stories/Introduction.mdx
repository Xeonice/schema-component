## 欢迎来到 Schema Component

Schema Component 是一个强大的、类型安全的 schema 驱动组件系统，专为现代 Web 应用设计。

### ✨ 核心特性

- **🔒 类型安全**: 完整的 TypeScript 类型推导
- **🔗 关联字段**: 支持 O2O、O2M、M2O、M2M 关系
- **✅ 强大验证**: 基于 Zod 的验证系统
- **🎨 主题系统**: 灵活的主题定制
- **⚡ 高性能**: 优化的渲染引擎
- **📦 模块化**: Monorepo 架构，按需使用

### 📦 包结构

本项目采用 monorepo 架构，包含以下核心包：

- **@schema-component/schema**: Schema 定义和验证
- **@schema-component/engine**: 渲染引擎
- **@schema-component/theme**: 主题系统
- **@schema-component/storybook**: 文档和示例（本站点）

### 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动 Storybook
pnpm --filter storybook storybook
```
